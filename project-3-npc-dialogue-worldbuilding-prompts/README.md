# NPC Dialogue & World-Building Prompt System

This project showcases a structured prompt framework for generating immersive NPC dialogue and consistent fantasy world lore.

## Objective
- Maintain character personality across dialogues
- Control tone, lore accuracy, and narrative depth
- Avoid generic or immersion-breaking responses

## Prompt Engineering Techniques Used
- Role conditioning
- World rule constraints
- Tone and personality anchoring
- Output formatting control

## Use Case
Game development, interactive storytelling, and AI-driven narrative systems.
